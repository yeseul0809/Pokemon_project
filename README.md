# 🔥 Pokemon project

## 🖥️프로젝트 소개

- 포켓몬 도감을 보여주는 웹사이트입니다.

<br>

## 🕰️개발 기간

- 24.07.02~ 07.04

<br>

## 📌기능

- 스크롤을 내릴때 포켓몬 리스트를 일정수만 보여주며 무한스크롤이 적용됩니다.
- 특정 포켓몬을 클릭시 디테일 페이지로 이동하여 상세정보를 보여줍니다.
- 선택된 포켓몬 캐릭터의 이름에 맞는 title 이 적용됩니다.
- 디테일 페이지에서 리스트로 돌아 갈 수 있습니다.

<br>

## 💠기술스택

- NEXT.JS
- TAILWINDCSS
- TYPESCRIPT
